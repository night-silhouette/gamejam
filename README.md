## 游戏立项——“概念” 
**子母牌**，母牌引出子牌。子牌又扶持加强母牌。 
玩家将在对局中体验到奇思妙想般的卡片效果与紧张刺激的对局比拼。 
## 立意阐述:
因为大赛的主题是桥梁，这是一个自然事物，如果是从自然熟悉入手是挖不出内涵的。内涵就是人赋予的社会
关系，桥梁落实到社会属性中就是强调一种连接。 
但是社会主义国家强调集体主义，从个人英雄主义入手不太好，所以说去传达一种脱离社会历史性的关系比较经久不衰。
## 游戏引导：
暂时只支持局域网联机，想要远程联机的可以尝试使用内网穿透
进入游戏后，Tab键切换视角，总共两个视角，一个侧视角，还有一个俯视角
侧视角选中卡牌出战，在俯视角使用a键攻击，使用d键释放技能，每回合谁出手由剪刀石头布决定（可手动，可系统自动）使用q键打开法术牌背包，点击法术牌可以使用。
击败对手所有的战斗牌即可获胜
